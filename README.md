# Astoria AI Studio

Astoria AI Studio - Передовой ИИ-стилист и инструмент для реставрации фото.

## Возможности

*   **Виртуальная укладка**: Примерьте сотни причёсок с помощью Gemini 2.5 Flash.
*   **HD Реставрация**: Удаление водяных знаков, логотипов и улучшение качества фото до 8K.
*   **Визуализация цвета**: Реалистичное окрашивание волос.
*   **Identity Lock**: Сохранение черт лица при любых изменениях.
